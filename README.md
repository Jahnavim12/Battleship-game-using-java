# ⚓ Battleship Game

A fully interactive Battleship game built using Java Swing with an AI-powered computer opponent and a modern naval-themed graphical interface.

---

## 📌 Features

- Interactive Battleship gameplay
- Computer AI opponent
- Java Swing graphical user interface
- Ship placement system
- Score and fleet tracking
- Hit, miss, and sunk indicators
- Auto ship placement
- Real-time battle mechanics

---

## 🛠️ Technologies Used

- Java
- Java Swing
- Object-Oriented Programming (OOP)

---

## 🚀 How to Run

### Compile the program

```bash
javac Battleship.java
```

### Run the game

```bash
java Battleship
```

---

## 🎯 Gameplay Instructions

- Place your ships on the grid
- Start the battle
- Click on enemy grid cells to attack
- Sink all enemy ships to win the game
